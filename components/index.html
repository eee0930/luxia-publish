Introduction

Getting Started

css, js include

src/scc/common dir에 있는 style.css와 src/js/common dir에 있는 app.js를 html 파일에 포함시켜줍니다.
example

<head>
  ...
  <link rel="stylesheet" href="src/css/common/style.css" />
</head>
<body>
  ...
  <script src="src/js/common/app.js"></script>
</body>
  1. style.css에는 모든 components의 style이 import되어 있습니다.
  2. app.js에는 모든 components의 인터렉션와 모션이 구현되어 있습니다.

Button

Components

button

button의 기본 스타일 가이드 입니다. button은 button 크기와 button 스타일 class를 삽입합니다.
view
example

<button type="button" class="btn btn-sm btn-primary">sm</button>
<button type="button" class="btn btn-md btn-primary">md</button>
<button type="button" class="btn btn-lg btn-primary">lg</button>
  1. 기본적으로 btn class를 삽입합니다.
  2. 버튼의 사이즈는 btn-sm, btn-md, btn-lg class를 삽입하여 조절합니다.
view
example

<button type="button" class="btn btn-md btn-primary">primary</button>
<button type="button" class="btn btn-md btn-secondary">secondary</button>
<button type="button" class="btn btn-md btn-tertiary">tertiary</button>
<button type="button" class="btn btn-md btn-cancel">cancel</button>
  1. 버튼의 스타일 및 사용성에 맞게 btn-primary, btn-secondary, btn-tertiary, btn-cancel class를 삽입합니다.
view
example

<button type="button" class="btn btn-md btn-primary btn-icon">icon markup</button>
<button type="button" class="btn btn-md btn-secondary btn-icon">icon markup</button>
<button type="button" class="btn btn-md btn-tertiary btn-icon">icon markup</button>
  1. 아이콘만 사용하는 버튼은 btn-icon class를 삽입합니다.
  2. 아이콘 버튼도 사이즈, 스타일 class를 넣어 스타일을 조정할 수 있습니다.
view
example

<button type="button" class="btn btn-md btn-lg-pc btn-primary">mo: md, pc: lg</button>
<button type="button" class="btn btn-md btn-sm-pc btn-primary">mo: md, pc: sm</button>
<button type="button" class="btn btn-md btn-primary">mo: md, pc: md</button>
  1. 모바일 버전과 pc 버전의 버튼 사이즈를 다르게 하고 싶다면 btn-사이즈-pc class를 삽입하면 됩니다.
  2. 위 예시 버튼들은 810이하 사이즈에서 모두 btn-md 사이즈와 같습니다.

text button

text 또는 text와 아이콘을 함께 사용하는 button style입니다.
view
example

<!-- text button -->
<button type="button" class="btn btn-lg btn-text">text</button>
<button type="button" class="btn btn-sm btn-text">text</button>

<!-- icon과 함께 -->
<button type="button" class="btn btn-lg btn-text">
  <span>text</span>
  <img src="" />
</button>
<button type="button" class="btn btn-sm btn-text">
  <span>text</span>
  <img src="" />
</button>
  1. text button의 사이즈는 btn-lg, btn-sm 두가지 class로 조정합니다.
  2. span tag 안에 text를 넣고 Icon과 함께 사용할 수 있습니다. icon은 img, svg 두가지만 사용 가능합니다.

download button

다양한 아이콘을 사용하여 download 버튼 같이 text와 event 버튼을 함께 노출할 수 있습니다.
view
downloadfile.txt
example

<div class="btn btn-download">
  <div class="text">
    <img src="assets/icons/clip.svg" />
    <span>downloadfile.txt</span>
  </div>
  <!-- 닫기 버튼 -->
  <button class="times">
    <img src="assets/icons/times.svg" />
  </button>
</div>
  1. btn-download class 안 times class를 가진 button을 클릭 시 버튼이 사라집니다. (app.js)
  2. download 버튼은 btn-download 안의 다양한 Element들에 click event가 발생할 수 있으므로 div를 상위 html로 마크업 할 것을 추천드립니다.

send button, pause button

view
example

<button type="button" class="btn btn-send">
  <span class="blind">send button</span>
</button>

<button type="button" class="btn btn-pause">
  <span class="blind">pause button</span>
</button>
  1. btn-send와 btn-pause class를 넣어 각각의 버튼을 만들 수 있습니다.
  2. 아이콘은 이미 style에 삽입되어 button 태그 안에 아무 것도 넣을 필요는 없지만 각각의 버튼 표시를 위해 blind class를 삽입한 span 태그로 각 버튼을 설명할 수 있습니다.
